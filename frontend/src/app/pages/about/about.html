Arquitetura

O sistema é dividido em três camadas independentes, cada uma com uma responsabilidade clara:

Angular 20 + Material Captura de webcam, indicadores em tempo real
↓ REST + WebSocket
Spring Boot 3 (Java 21) Usuários, sessões, métricas, orquestração
↓ REST
FastAPI + OpenCV + MediaPipe (Python 3.12) Análise facial e prova de vida

Tecnologias

  • Backend: Java 21, Spring Boot 3, Spring Data JPA, PostgreSQL, WebSocket
  • Frontend: Angular 20, Angular Material, RxJS
  • Visão computacional: Python 3.12, FastAPI, OpenCV, MediaPipe, NumPy
  • Infraestrutura: Docker, Docker Compose

Por que não simulamos autenticação

Este assistente não substitui nem contorna sistemas de reconhecimento facial — ele apenas avalia a qualidade técnica da captura (iluminação, nitidez, enquadramento, estabilidade) e conduz uma prova de vida baseada em geometria real do rosto, reduzindo falsos negativos causados por más condições de captura antes que o usuário envie a imagem para validação em outro sistema.
